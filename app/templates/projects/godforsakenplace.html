Online Mafia Game

A multiplayer online adaptation of the classic Mafia card game, featuring real-time gameplay, immersive storytelling, and seamless social interaction.

Features

  • Real-time multiplayer gameplay with roles like Mafia, Detective, and Citizen.
  • Lobby system with game codes for easy access and team management.
  • Role-based actions for immersive night and day cycles.
  • Customizable characters with unique abilities to enrich gameplay.
  • Secure authentication with options for guest play or registered accounts.

Technologies Used

  • Frontend: React for dynamic and responsive UI.
  • Backend: .NET Core 7 with ASP.NET MVC for secure and scalable server-side logic.
  • Database: Azure-hosted database for storing game data and user profiles.
  • Real-Time Communication: SignalR for chat and in-game updates.
  • Media: DALL-E for character illustrations, ensuring style consistency.

Challenges & Solutions

One significant challenge was ensuring real-time communication and game synchronization. This was addressed using SignalR for efficient two-way communication between players and the server. Additionally, styling consistency and accessible navigation were achieved with centralized CSS files.

Future Enhancements

  • Enhanced AI-generated story narration using ChatGPT with text-to-speech integration.
  • Additional characters with unique abilities for more varied gameplay.
  • Improved mobile responsiveness and optimized game performance for larger player groups.
