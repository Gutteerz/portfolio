Portfolio Website

A modern, responsive portfolio website designed to showcase projects, skills, and accomplishments, built using Flask, HTML/CSS, and JavaScript.

Features

  • Dynamic project filtering and categorization.
  • Responsive design for seamless viewing across devices.
  • Dedicated sections for about, contact, and individual project details.
  • Fully CI / CD wokflow for autonomus deployment
  • Custom styling with modular CSS files for easy scalability.

Technologies Used

  • Framework: Flask
  • Frontend: HTML, CSS, JavaScript
  • Hosting: DigitalOcean
  • Secret Manager: Github

Challenges & Solutions

One challenge was ensuring smooth filtering for dynamic project categories. This was addressed with modular JavaScript to manage filtering logic effectively.

Future Enhancements

  • Integrating a blog section to share updates and insights.
  • Adding a dark mode toggle for better accessibility.
  • Improving performance with image optimization and caching.
