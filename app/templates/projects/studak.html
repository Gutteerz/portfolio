Student Activity App

A user-friendly mobile application designed to help students discover, organize, and participate in activities within their budget and group size.

Features

  • Activity recommendations based on budget, group size, and location.
  • Integrated calendar with the ability to schedule activities with friends.
  • User profile creation with friend lists and invitations.
  • Real-time activity filtering and results fetched from Firestore database.
  • Secure authentication via Google Firebase.

Technologies Used

  • Frontend Framework: React Native with Expo
  • Backend: Firebase for authentication and database
  • Navigation: React Navigation
  • Database: Firestore
