Mario Game

A 2D platformer inspired by the classic Mario series, featuring custom levels, retro graphics, and immersive gameplay.

Features

  • Classic platforming mechanics: jumping, collecting coins, and defeating enemies.
  • Custom-designed levels with increasing difficulty.
  • Smooth animations and responsive controls.

Technologies Used

  • Programming Language: C++
  • Game Engine: SDL2
  • Graphics: Custom sprites and tiles

Challenges & Solutions

Implementing efficient collision detection was a major challenge. This was resolved by using a spatial grid approach, optimizing performance and ensuring smooth gameplay.
