RTOS with bitmask communication

A real-time operating system (RTOS) with bitmask communication for efficient task management and data exchange. For use in ROV with limited resources.

Features

  • Lightweight and efficient design for microcontroller deployment.
  • Acts like a central point for multiple sensors for sending to topside
  • Main comunication between topside and ROV for all comunications like sensors, joysticks and diagnostics

Technologies Used

  • RTOS: ZephyrOS
  • Language: C
  • Hardware: Nucleo F767zi
  • Link: Ethernet
