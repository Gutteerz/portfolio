Smart Clock

A feature-rich smart clock powered by an STM32 microcontroller, designed for enhanced time management, weather updates, and user interactivity.

Features

  • IP-based geolocation to fetch Unix epoch time, time zone, daylight saving data, and location details at startup.
  • Displays current date, time, and day of the week with real-time updates.
  • Allows users to set recurring alarms with snooze, mute, and enable/disable functionalities.
  • Shows room temperature and humidity updated regularly.
  • Provides weather forecasts fetched from an API, with updates every 15 minutes.
  • Enables user to input city name or precise coordinates for accurate weather forecasts.
  • Displays top news headlines fetched via RSS feeds with smooth scrolling text.

Technologies Used

  • Microcontroller: STM32
  • Display: DFRobot LCD
  • Data Fetching: APIs for geolocation, weather, and RSS feeds
  • Programming Language: C/C++

Challenges & Solutions

The main challenges included handling dynamic data updates and ensuring accurate geolocation. Aswell as cleaning the RSS feed to ensure html tags was not included as a part of the news feed.
