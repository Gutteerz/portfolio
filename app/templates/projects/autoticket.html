AI Dashcam

An AI-powered system for automatic traffic sign detection and recognition, designed for scalability, efficiency, and seamless integration with dashcams and other video feeds.

Features

  • Real-time traffic sign detection and recognition using YOLOv11.
  • Lightweight and efficient design for potential edge deployment.
  • Supports advanced functionalities like cruise control and speed warnings.

Technologies Used

  • AI Model: YOLOv11 with custom fine-tuned weights
  • Backend: FastAPI
  • UI: HTML, CSS, and JavaScript
  • Data Labeling: Label Studio

Challenges & Solutions

We faced issues with data consistency and labeling accuracy. These were mitigated by refining the dataset and using Label Studio for structured labeling. Scalability and noise reduction were also addressed during model fine-tuning.
